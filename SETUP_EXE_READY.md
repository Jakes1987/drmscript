## What Happens After Installation

### User's Computer Has:
1. **Kayo Script Gen folder** in AppData/Local
   - Location: C:\Users\[Username]\AppData\Local\Kayo Script Gen
   - Size: ~13 MB
   - Contains: Main application + configuration

2. **Desktop shortcut**
   - "Kayo Script Gen" icon on desktop
   - Double-click to launch

3. **Start Menu entry**
   - Windows Start Menu > Kayo Script Gen
   - Also searchable in Windows

4. **Uninstaller**
   - Can uninstall via Windows Add/Remove Programs
   - Or run Uninstall.bat from installation folder

### User Interaction Flow:
```
1. Launch (desktop shortcut or Start Menu)
   ↓
2. Web GUI opens (browser window)
   ↓
3. Enter Kayo credentials
   ↓
4. Test credentials
   ↓
5. Configure settings
   ↓
6. Generate script
   ↓
7. Download files
   ↓
8. Deploy to O11V4
   ↓
9. Done!
```
