WHAT IS THIS?
=============

This is an installer for the Kayo to O11V4 Script Generator - a professional
tool that generates provider scripts for the O11V4 video streaming platform
using your real Kayo account.

NO TECHNICAL KNOWLEDGE REQUIRED - just follow the simple steps below!


SYSTEM REQUIREMENTS
===================

✓ Windows 7, 8, 10, or 11
✓ 2 GB RAM minimum (4 GB recommended)
✓ 200 MB free disk space
✓ Internet connection
✗ NO administrator rights needed!


QUICK START (4 STEPS)
=====================

STEP 1: Run the Installer
─────────────────────────
1. Find the file: setup.exe
2. Double-click the setup.exe file
3. A window will open asking about the installation
   
   The installer will ask you to click "Install"


STEP 2: Wait for Installation
──────────────────────────────
A progress window will show 4 steps:
  [1/4] Creating directories... 
  [2/4] Copying files...
  [3/4] Creating shortcuts...
  [4/4] Creating uninstaller...

This takes about 30-60 seconds. Just wait!


STEP 3: Installation Complete
──────────────────────────────
When you see "✓ Installation Complete!" message:
  - The application is installed
  - Desktop shortcut created
  - Start Menu link created
  - You can click "Close" or press any key


STEP 4: Launch the Application
───────────────────────────────
After installation, use ONE of these methods:

METHOD A (Easiest): Click Desktop Shortcut
  - Look for "Kayo Script Gen" icon on your desktop
  - Double-click it
  - The application opens in your browser

METHOD B: Use Start Menu
  - Click Windows Start button
  - Type: "Kayo Script Gen"
  - Click the application to launch

METHOD C: Manual File Browser
  - Open File Explorer
  - Go to: C:\Users\[YourUsername]\AppData\Local\Kayo Script Gen
  - Double-click: KayoScriptGen.exe


USING THE APPLICATION
=====================

Once launched, you'll see a web interface with these steps:

1. ENTER CREDENTIALS
   - Email: Your Kayo account email
   - Password: Your Kayo account password
   
   (Your credentials are NOT stored in generated scripts)

2. TEST CREDENTIALS
   - Click the "Test Credentials" button
   - Wait for verification (5-10 seconds)
   - You'll see ✓ if credentials are valid

3. CONFIGURE SETTINGS
   - Provider Name: Name for your O11V4 provider
   - Select channels and events you want to include
   - Choose quality settings

4. GENERATE
   - Click "Generate" button
   - Wait for script generation (10-30 seconds)
   - You'll see a download button when ready

5. DOWNLOAD
   - Click "Download" to save your provider files
   - Two files will download:
     a) provider_name.py (the actual script)
     b) provider_name.json (configuration)

6. DEPLOY TO O11V4
   - Follow your O11V4 documentation
   - Upload the generated .py file
   - Deploy and enjoy!


TROUBLESHOOTING
===============

PROBLEM: "Application won't start"
─────────────────────────────────
Solution A: Try the desktop shortcut again
Solution B: Go to Start Menu, search "Kayo Script Gen"
Solution C: Check that Windows 7+ is installed (windows key + R, type "winver")

PROBLEM: "Credentials fail / won't authenticate"
──────────────────────────────────────────────────
Check 1: Is your Kayo account active?
         - Try logging into https://www.kayosports.com.au
         - If that works, your account is active

Check 2: Is password correct?
         - Try logging into Kayo website first
         - Make sure CAPS LOCK is off

Check 3: Does Kayo have content?
         - Your account must have access to Kayo content
         - Some regions may have limited content

PROBLEM: "Port already in use" error
──────────────────────────────────────
Solution: This is very rare. Wait 30 seconds and try again.

PROBLEM: Application launches but can't connect
─────────────────────────────────────────────────
Check that:
  - Internet is working
  - No firewall blocking port 5000
  - Kayo API is accessible (check https://www.kayosports.com.au)

PROBLEM: Download doesn't work
────────────────────────────────
Solution A: Check browser download settings
Solution B: Make sure you have write permissions to Downloads folder
Solution C: Try a different web browser (Chrome, Edge, Firefox)


UNINSTALL INSTRUCTIONS
======================

To remove Kayo Script Generator:

METHOD A (Easiest): Uninstall from Windows
────────────────────────────────────────────
1. Click Windows Start button
2. Type: "Add or remove programs"
3. Search for "Kayo Script Generator"
4. Click "Uninstall"
5. Confirm when asked

METHOD B: Manual Uninstall
──────────────────────────
1. Go to: C:\Users\[YourUsername]\AppData\Local\Kayo Script Gen
2. Look for file: Uninstall.bat
3. Double-click Uninstall.bat
4. Click OK when complete

METHOD C: Manual File Deletion
───────────────────────────────
1. Delete folder: C:\Users\[YourUsername]\AppData\Local\Kayo Script Gen
2. Delete shortcut: C:\Users\[YourUsername]\Desktop\Kayo Script Gen.lnk
3. Delete Start Menu link:
   C:\Users\[YourUsername]\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Kayo Script Gen
