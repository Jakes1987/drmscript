## How It Works in O11V4

1. IMPORT FILES
   ├─ Copy kayo_demo.py → /path/to/o11v4/scripts/
   └─ Copy kayo_demo.json → /path/to/o11v4/providers/

2. RESTART O11V4
   └─ O11V4 scans providers/ and loads Kayo as available provider

3. CONFIGURE
   ├─ Go to Config tab in web UI
   ├─ Select "Kayo" provider
   ├─ Set up network parameters if needed
   └─ Enable channels for streaming

4. STREAM
   ├─ Click channel in web UI
   ├─ O11V4 calls: kayo_demo.py --action manifest --id channel_id
   ├─ Script returns manifest URL + headers
   └─ O11V4 begins streaming with HLS/SRT output

5. BACKGROUND OPERATIONS
   ├─ Token refresh happens automatically (before expiry)
   ├─ Heartbeats sent on interval to keep session alive
   ├─ On-demand streams start/stop as needed
   └─ Error recovery with automatic retry
