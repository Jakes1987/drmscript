# Quick Start - GitHub & Ubuntu Setup

## Your Next Steps

### Step 1: Create GitHub Repository
1. Go to: https://github.com/new
2. Create repository named: `kayo-script-gen`
3. Select: **Public**
4. Copy your repository URL

### Step 2: Push Your Code

Open PowerShell in your project folder and run:

```powershell
# Add GitHub remote
git remote add origin https://github.com/Jakes1987/drmscript

# Push to GitHub
git add .
git commit -m "Initial commit"
git push -u origin main
```

### Step 3: Update Installer Scripts

Edit these files and replace `yourusername` with YOUR username:
- `install-ubuntu.sh`
- `install-one-liner.sh`

Then push the changes:
```powershell
git add install-ubuntu.sh install-one-liner.sh
git commit -m "Update installer URLs"
git push
```

## What Ubuntu Users Will See

## Installation Locations (Ubuntu)

After installation, users will have:
- **Application**: `~/.local/kayo-script-gen/`
- **Command**: `~/.local/bin/kayo-script-gen`
- **Desktop Entry**: Application menu

1. ✅ Downloads project from GitHub

3. ✅ Installs dependencies via pip3

5. ✅ Creates launcher command
